# Smart AI Study Planner

## Overview
This project helps students create and analyze study plans.

## Features
- Create Study Plan
- Save Data in CSV
- Analyze Subjects
- Detect Weak Subject
- Generate Study Graph

## Technologies Used
- Python
- Pandas
- Matplotlib
